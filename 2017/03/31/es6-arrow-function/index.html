    Arrow function

    箭头函数表达式具有比函数表达式更短的语法，并且不绑定其自身的this，arguments，super或new.target。箭头函数表达式最适合于非方法函数，它们不能用于构造函数。

    Shorter functions

    函数较短

    No binding of this

    不绑定this(自己的理解, 不将this绑定到所使用的上下文中): 函数体内的this对象，就是定义时所在的对象，而不是使用时所在的对象
